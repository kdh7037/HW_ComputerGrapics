컴퓨터그래픽스 과제2 - 2015920005 김대현

동별, 위치를 이용하여 웹사이트 2개 제작. (웹사이트 폴더는 편의를 위해 기존에 있던 population, wifi를 그대로 사용하였습니다.)

개발환경 ( visual studio code, 네이버 Whale 브라우저 )

1. 동별 웹사이트 제작
사이트 제목 : 서울시 고령화인구(동별) 시각화
사용한 자료 : 고령화, 독거노인 인구 정보

레이어의 높이 : 총 인구수
레이어의 색 : 고령화 인구의 비율, 고령화 비율이 높을 수록 진한 색
레이어의 선의 색 : 고령화 인구의 수, 색이 붉을 수록 고령화 인구가 많다.

레이어의 색은 다른 이쁜 색으로 넣으려 하였으나, 레이어의 선이 육안으로 구별이 어려웠기 때문에, 흑백으로 설정하였습니다.
레이어의 선은 굵게 만들어서 육안으로 쉽게 보이게 하려 했으나, 레이어에서 선을 육안으로 지원을 하지 않았습니다.

중간중간 비는 동이 존재합니다. 각기 다른 동별 자료를 활용하였기 때문에, 자료가 다를 경우에 시각화에 문제가 생겼기 때문에, 문제가 생기는 동을 제거하였습니다. 동을 확인하여 수정 작업을 거쳤어야 했으나 역량 부족으로 진행하지 못했습니다. 죄송합니다.

2. 위치 웹사이트 제작
사이트 제목 : 서울시 소방관련 위치정보
사용한 자료 : 비상소화장치, 서소위치, 소방대상물, 소화용수 위치 정보

소방대상물은 붉은색 HexagonLayer를 활용하여 바닥의 육각형 색으로 확인 가능
소화용수는 푸른색 HexagonLayer를 활용하여 육각형의 가는 기둥으로 확인 가능
비상소화장치와 서소위치는 IconLayer를 활용하여 확인 가능
(비상소화장치 붉은색, 서소위치 초록색)

소방대상물과 소화용수는 각각 툴팁이 다르게 적용됩니다.
